# PhpSoftBox ORM

ORM компонент для PhpSoftBox. Работает поверх `phpsoftbox/database`.

> Статус: beta

> TODO: сейчас ORM подтягивает пакет `phpsoftbox/inflector` как реальную зависимость.

> чтобы можно было подключать любой инфлектор без установки вcего `phpsoftbox/inflector`.
